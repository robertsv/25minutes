What is it?

25minutes is a simple yet very powerful time tracking tool. It is based on Pomodoro Technique. The idea is simple - you work for given time (25 or 45 or any other time span that fits you) then you take a break and after break you start again to work.

How to use it?

By default work time is 25 minutes and break is 5 minutes. In settings you can change it. After each session (work and break session) there is a notification. Next session starts automatically but you can pause it or restart.

Built with Angular 8 and Boostrap 4.4, source is on GitHub.
